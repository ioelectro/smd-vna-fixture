# SMD VNA Fixture

A compact, open-source PCB fixture for measuring SMD passive components with a Vector Network Analyzer (VNA).

The fixture is designed to minimize parasitic inductance and capacitance, making it suitable for accurate impedance measurements over a wide frequency range. It supports quick soldering of SMD components and is ideal for characterizing capacitors, inductors, ferrite beads, and other two-terminal devices.

Features
Optimized PCB layout for RF measurements
Suitable for one-port (S11) measurements
Low parasitic fixture design
Easy SMD component replacement
Open-source Altium Designer project
Ready for fabrication with standard PCB manufacturers
Applications
Measuring SMD capacitors and inductors
ESR and impedance analysis
Resonant frequency characterization
RF component evaluation
Educational and laboratory use
Notes
For best accuracy, perform VNA calibration at the fixture reference plane.
Use high-quality SMA connectors and short RF paths.
Measurement accuracy depends on VNA calibration quality and fixture construction.
